# Unity Engine 6 Overview and Installation

## Project Name
My project-1

## Unity Version Used
Unity 6.2 (6000.2.10f1)

## Overview
This task involved installing Unity Hub, setting up Unity Engine 6, and creating a new project. The Unity interface was explored to understand its main components such as Scene View, Hierarchy, Inspector, Project, and Console windows.

## First Impressions
Unity provides a well-structured and powerful interface for game development. The layout is organized, making it easy to manage game objects and assets. Initially, it may feel complex, but after exploring the different panels, it becomes more intuitive. The real-time Scene and Game views are especially useful for visualizing changes instantly.
